🧬 PHARMA-AI

AI-Driven Pharmaceutical Formulation Optimizer

📋 Quick Start Guide:

  1. Start Backend: Open terminal in backend folder and run:
    python -m uvicorn server:app --reload --port 8001
  2. Open Frontend: Click the button below
🚀 Launch PHARMA-AI
⚠️ Important: Make sure the backend server is running on port 8001 before using the app!

Backend API: http://localhost:8001

Frontend Build: ./frontend/build/
